# GP Scoring System

## Description
Gold piece (GP) scoring system tracking the player's earnings from bumper hits.

## Implementation Details
- `src/state/gameState.ts` - GP tracking in game state
- `src/hooks/useGameState.ts` - addGp action
- `src/components/HUD.tsx` - GP display with formatted numbers
- Starting GP: 500
